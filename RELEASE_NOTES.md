# v0.1.0 首次开源发布

Codex 额度雷达的第一个公开版本，面向 Windows Codex 桌面端。

## 功能亮点

- 预计全体额度刷新日期、时间和倒计时
- 本机 Codex 剩余额度与官方重置时间
- Tibo 公开公告跟踪、中文翻译和本机 Codex 分析
- 主面板、右侧窄轨、最小浮标三种形态
- 只在 Codex 前台时显示
- 自由拖动、位置记忆、展开/收起顺滑动画
- 新公告 Windows 通知与本地去重
- 登录启动监听和离线缓存

## 当前限制

- 仅支持 Windows。
- 全体刷新时间属于历史统计预测，不是官方时间。
- 依赖公开追踪页和 Codex 本地接口，外部结构变化可能导致读取失败。

## 安装包

下载 Release 中的 `codex-reset-watcher-v0.1.0.zip`，解压后按照 README 的“安装与运行”操作。
